basics: creation and element access

The elements of an array can be any object class and can even be mixed within the array.

numbers  = [1,2,3,4]

strings  = ["foo", "baz", "boo"]

empty_ar = []

empty_ar = Array.new

game_board = Array.new(8) { Array. new(8) }       #init an 8 x 8 multi-dim array this way.

 

ar[i]                         #=> ith element

ar[-2]                        #=> 2nd from end

ar[1..3]                      #=> [6,7,8]

ar[2,2]                       #=> [7,8]

 

<<               shovel operator

Appends the element or array on the right to the end of the array on the left.

Returns: the array on the left, modified.

let:       ar = [5, 6, 7]

          ar << 9

returns:    [5, 6, 7, 9 ]

 

let:       ar = [5, 6, 7]

          ar << [ 1, 2, 3 ]

returns:    [5, 6, 7, 8, [ 1, 2, 3 ] ]

+                plus

Appends (concatenates) the elements of arrays together.

See also .concat(other_array)

the expression:    [5] + [ 1, 2, 3] + [ 10, 12]

returns:          [5, 1, 2, 3, 10, 12]

-                 minus, subtract

Removes all occurrences of the elements of the second operand.

the expression:    [5, 6, 7, 6, 7] - [6, 7 ]

returns:          [5]

*                 array multiply

Combines copies of the array into a new array.

let:       ar = ["Ouch!"]

          ar * 3

returns:   ["Ouch!, Ouch!, Ouch!"]

all? { |el| block_code }

Returns true if all elements of the array pass the block's test.

let:       ar = [5, 6, 7]

          ar.all?{ |el| el > 4 }

returns:  true

any? { |el| block_code }

Returns true if any element of the array passes the block's test.

let:       ar = [5, 6, 7]

          ar.any?{ |el| el > 6 }

returns:   true

Adds (concatenates) the elements of other_array onto the end of the array.
Returns: the array is modified.

Leaves the original array unmodified.

each { |el| block_code }

Performs the block_code on each element of the array. Original array is not modified, thus it's is all about the side effects. 

Returns the unmodified array.

let:       ar = ["Joe", "Fred", "Sally"]

          ar.each{ |el| puts "Hi " + el +"!"}

StdOut:

          Hi Joe!

          Hi Fred!

          Hi Sally!

 

returns:   ["Joe", "Fred", "Sally"]

          (the unmodified array)

 

each_with_index { |el, index| block_code }

Performs the block_code on each element of the array.

Passes the element and its index to the block.

Original array is not modified, thus it's is all about the side effects. 

Returns the unmodified array.

let:       ar = ["Joe", "Fred", "Sally"]

          ar.each{ |el| puts "Element: " + index +" is " + el"}

StdOut:

          1: Joe

          2: Fred

          3: Sally

 

returns:   ["Joe", "Fred", "Sally"]

          (the unmodified array)

flatten
flatten (n)
flatten!
flatten! (n)

Without a parameter, converts an array of arrays into a one dimensional array, inserting all elements of the embedded arrays at the top level. 

With parameter n, flattens the array n levels

Returns: a new flattened array.

Returns!: the ! version returns the modified array or nil if no modifications were made.

let:       ar = [[1,2], [35, 17, ["Joe", "Fred"]]]

returns:   [1, 2, 35, 17, "Joe", "Fred"]

          (a new array)

 

let:       ar = [[1,2], [35, 17, ["Joe", "Fred"]]]

           ar.flatten!(1)

returns:   [1, 2, 35, 17, ["Joe", "Fred"]]

          (the array, modified)

inject(sym)

sym is an operator, for example, addition, written as :+ . inject will return the value of the statement formed by "injecting" the operator between each element in the array. For example if the sym is :+ inject will perform el1 + el2 + el3... That is, it will return the sum of the elements.

inject(initial, sym)

Same as above but include initial as the starting value.

inject (initial) { |memo, element| block_code }

Calls block_code, passing memo and element at each iteration. block_code should use memo in its calculation. After each iteration, the value returned by block_code is set to memo. Then memo and the next element are passed to block_code.

memo is a traditional name for an accumulated value. If initial is not present, the first element of the array is used instead.

let:       ar =  ar = [1, 2, 3, 4]

          ar.inject(:+)

returns:   10

           (inject computes: 1 + 2 + 3 + 4 )

let:       ar = [1, 2, 3, 4]

          ar.inject(25,:+)

returns:   35

           (inject computes: 25 + 1 + 2 + 3 + 4 )

let:       ar = [1, 2, 3, 4]

          ar.inject(10) { |memo, element| memo + element}

returns:   20

max

Returns: the maximum value of the array. Order is determined by using the <=> operator.


max  { |a, b| block_code }

Provide your own comparison test. Calls block_code on pairs of elements of the array. Your block should return -1, 0, 1 if a is <, ==, or > b respectively.

Returns: the maximum value of the array.

let:       ar = ["joe", "fred", "sally"]

          ar.max

returns:   ["sally"]

           (the original array is unmodified)

let:       ar = [2, 1, -10, 9]

          ar.max{|a, b| a - b }

returns:   9

           (the original array is unmodified)

pop

Returns: the last element from the array. The original array is modified. If array is empty returns nil.


pop (n).

Returns: a new array containing the last n elements of the original. The original array is modified. If the original array is empty, returns [].

 

let:       ar = ["Joe", "Fred", "Sally"]

          ar.pop

returns:   "Sally"

          (ar is now ["Joe", "Fred"])

let:       ar = [1, 2, 3, 4]

          ar.pop(3)

returns:   [2, 3, 4]

          (ar is now [1])

sample
sample (n)

sample returns an element chosen at random from the array or nil if the array was empty.
sample(n) returns a new array containing a random selection of n elements. There will be no duplicates unless the original array contains duplicates itself. If the array is empty, it returns an empty array..

The original array is unmodified.

let:       ar = [1, 2, 3, 4, 5, 6]

          ar.sample

returns:   3

           (the original array is unmodified)

let:       ar = [1, 2, 3, 4, 5, 6]

          ar.sample(3)

returns:   [3, 1, 4]

           (the original array is unmodified)

 let:      ar = ["Joe", "Fred", "Sally"]

In both cases the original array is modified.

          ar.shift(3)

          (ar is now [4])

slice (index)

Delete the indicated element from the array.

Returns: the deleted element or nil if there is no index'th element.

slice (index, length)

Delete length elements starting at index from the array.

Returns: an array of the deleted elements. If there are not enough elements in the array, fewer than length elements will be returned.

slice (range)

Delete the range of elements from the array.

Returns: an array of the deleted elements. If both indices of the range are outside the array, return nil. If only the second index is outside then return as many as possible.

 

let:       ar = ["Joe", "Fred", "Sally"]

          ar.slice(1)

returns:   "Fred"

          (ar is now ["Joe", "Sally"])

let:       ar = [1, 2, 3, 4, 5, 6, 7, 8, 9]

          ar.slice(3, 4)

returns:   [4, 5, 6, 7]

          (ar is now [1, 2, 3, 8, 9])

let:       ar = [1, 2, 3, 4, 5, 6, 7, 8, 9]

          ar.slice(0..3)

returns:   [1, 2, 3, 4]

          (ar is now [5, 6, 7, 8, 9])

 

 

sort

Returns: a new sorted array. Order is determined by using the <=> operator.


sort  { |a, b| block_code }

Provide your own comparison test. Calls block_code on pairs of elements of the array. Your block should return -1, 0, 1 if a is <, ==, or > b respectively.

Returns: a new sorted array.


sort!
sort!  { |a, b| block_code }

Same as above but sorts the original array.

let:       ar = ["joe", "fred", "sally"]

          ar.sort

returns:   ["fred", "joe", "sally"]

           (the original array is unmodified)

let:       ar = [2, 1, 0, 9]

          ar.sort{|a, b| a - b }

returns:   [0, 1, 2, 9]

           (sort! modifies the original)

 

 

take (n)

Copy n elements from the beginning of the array

Returns: a new array containing the elements copied.

 

let:       ar = [1, 2, 3, 4, 5, 6, 7, 8]

          ar.take(5)

returns:  [1, 2, 3, 4, 5]

          (original array unmodified)

 

take_while { |el| block_code }

Passes elements from the beginning of the array to the code block and as long as the code block doesn't return false or nil, copies those elements into a new array

Returns: the elements copied from the beginning of the original array.

 

let:       ar = [1, 2, 3, 4, 5, 6, 7, 8]

          ar.take_while{|el| el < 6 }

returns:  [1, 2, 3, 4, 5]

          (original array unmodified)

 

uniq
uniq { |el| block_code }
uniq!
uniq! { |el| block_code }

Removes duplicate elements from array. If a block is given, uniq will call the block before comparing: block.call(element) <=> block.call(other_element)

 

Returns: uniq returns a new array, the ! versions modify the original.

 

let:       ar = [1, 2, 6, 4, 8, 6, 7, 8]

          ar.uniq

returns:  [1, 2, 6, 4, 8, 7]

          (original array unmodified)

let:       ar = [1, 4, 5, -4, 7]

          ar.uniq{|el| el.abs }

returns:  [1, 4, 5, 7]

          (original array unmodified)
