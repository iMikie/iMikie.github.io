As my career has progressed I've moved more and more into management, money raising, and sales.  Yet I still find myself asking, "can you do X?" and I'm still just as unwilling to accept "no" as an answer.  The greatest joy in Tech is wondering if something is possible and then demonstrating it is. 

 

So this year I'm getting a tune-up.  I've enrolled in Dev BootCamp, one of the growing number of insanely intensive trade schools that teach Web or Mobile technologies in 9 or 18 weeks. 

 

• What's it like to stop mid-career and learn with the kids?

• How have things changed at the ground level?

• How have they stayed the same?

 

Some of these posts will be required by Dev BootCamp but I'm hoping to add some value with my own reflections on the journey.  And a bunch of free code.
