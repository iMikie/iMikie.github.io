Spat operator

The splat operator turns a parameter list into an array:

def my_max(*values)
  values.max
end

max = my_max(3,4,5,2,1) 		#splat puts the parameters into array values.

And an array into a parameter list:

parms = [25, 5]

def my_divide(*[numerator, denominator)]
  numerator/denominator
end

my_divide(*parms) 		#splat turns the array [25,5] into a list of parameters.
